# GameArena — Gaming Website

A responsive gaming website built with **HTML**, **CSS**, and **Bootstrap 5**.

## Completed Sections

### Navbar
- Sticky top navigation with logo, nav links, notification icons, and a Sign In button
- Responsive hamburger menu on mobile with animated open/close icon
- Smooth scroll links to all sections

### Hero Section
- Full-screen hero with background image and dark overlay
- Large animated title with lime green accent color
- Description text and two CTA buttons (Start Gaming Now / Watch Tournaments)
- Neon-bordered hero image with a floating Active Players badge
- Fully responsive: stacked layout on tablet and mobile with scaled typography

### Our Games
- "Our Latest Games" carousel showcasing 9 games
- Three responsive carousels based on screen size:
  - **Mobile** (< 768px): 1 card per slide with swipe support
  - **Tablet** (768px – 991px): 2 cards per slide
  - **Desktop** (≥ 992px): 3 cards per slide
- Card hover effect with overlay, badge, and category label

## Tech Stack

- HTML5
- CSS3 (custom properties, keyframe animations, media queries)
- Bootstrap 5
- Font Awesome 6
- Google Fonts (Chakra Petch, Days One, Orbitron)
